# Touchbistro Test

## Technologies

 - **Root**: Lerna (monorepo manager, please run all commands from root folder);
 - **Front-end**: React, Next.js, Styled JSX, Jest, Enzyme, Axios;
 - **Back-end**: Express, Mocha, Nodemon;

## Installation
	
    npm install

## Running (Dev Mode)

    npm run dev

## Running (Prod Mode)

    npm run build && npm start

## Testing

    npm test
